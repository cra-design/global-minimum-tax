Filing a return

Filing details coming in Fall 2025

Information on how to file the forms in the return will be available in Fall 2025.

The first year of application of the Global Minimum Tax Act (GMTA) applies to qualifying multinational enterprise (MNE) groups for fiscal years that begin on or after December 31, 2023.

On this page

  • Forms to file in your return
  • Who must file
  • Who can file on your behalf
  • Before you file
  • When to file

Forms to file in your return

The member of a qualifying multinational enterprise group must file one or more of 3 forms. To learn specific requirements for filing each form, refer to:

  • Filing a GloBE Information Return (GIR)
  • Filing a Global Minimum Tax Return
  • Filing a GIR notification

Who must file

A member of a qualifying MNE group must file for a particular fiscal year when the group has a constituent entity located in Canada for the fiscal year.

What is a qualifying MNE group

A qualifying MNE group is an MNE group with €750 million or more in revenue reported in the consolidated financial statements of its ultimate parent entity (UPE) (in at least 2 of the 4 fiscal years immediately preceding the particular fiscal year), and is not composed exclusively of excluded entities for the particular fiscal year.

To find out who must file each form, refer to:

  • Who must file a GIR
  • Who must file a Global Minimum Tax Return
  • Who must file a GIR notification

    • Who can file on your behalf

    A constituent entity or person may be appointed to file on behalf of all constituent entities or persons that must file with the CRA.

    To find out who can file each form on your behalf, refer to:

  • Who can file a GIR on your behalf
  • Who can file a Global Minimum Tax Return on your behalf
  • Who can file a GIR notification on your behalf

    To find out what information you must gather to file each form, refer to:

  • Before you file a GIR
  • Before you file a Global Minimum Tax Return
  • Before you file a filing a GIR notification

    • When to file

    The deadline by which the forms must be filed for affected fiscal filing periods ending December 31, 2024, or prior is June 30, 2026.

    However, depending on a qualifying MNE group’s fiscal year, one of the following later dates is the due date for a qualifying MNE group’s forms:

    • 18 months after the last day of the fiscal year, if it is the first fiscal year that an entity of the qualifying MNE group is subject to:
      • Part 3 of the Act
      • a qualified Income Inclusion Rule (IIR)
      • a qualified Undertaxed Profit Rule (UTPR)
    • 15 months after the last day of the fiscal year, in any other case

    Qualified Income Inclusion Rule

    A qualified Income Inclusion Rule (IIR) for a fiscal year means an IIR that has qualified status (including transitional qualified status) for the fiscal year as determined by the Inclusive Framework and published on the website of the OECD.

    Qualified Undertaxed Profit Rule

    A qualified Undertaxed Profit Rule (UTPR), for a fiscal year, means a UTPR that has qualified status (including transitional qualified status) for the fiscal year as determined by the Inclusive Framework and published on the website of the Organisation for Economic Co-operation and Development.

    Qualifying foreign filing entity

    A qualifying foreign filing entity (QFFE) is an ultimate parent entity or a designated filing entity of the MNE group that is located in a foreign jurisdiction and that is obligated to file a GloBE information return with the tax authority of the foreign jurisdiction, provided that the foreign jurisdiction has a qualifying competent authority agreement that is in effect on or before the GloBE information return exchange date for the fiscal year. Canada must be a party to that agreement for the entity to be a QFFE.
